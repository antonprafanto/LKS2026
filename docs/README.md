## Alur cepat Modul E + Log Run

```
1. Baca PANDUAN-MODUL-E.md (Bagian 1–2)
2. Isi Undian Kubus (marker baris 24 + kubus baris 28–36)
3. Cek Modul E otomatis → isi Hasil Run kolom H setelah run
4. Isi Log Run (PANDUAN-LOG-RUN.md Bagian 4–7)
5. Tanda tangan juri + peserta
```
